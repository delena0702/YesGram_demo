YesGram, Image to Nonogram Puzzle Converter

나의 사진으로 만드는 하나뿐인 노노그램 퍼즐
사용자의 이미지를 노노그램 퍼즐로 변환할 수 있습니다.

Generate Puzzle

퍼즐 생성

사용자가 업로드한 이미지를 OpenCV를 활용한 전처리 과정을 거쳐 해가 유일한 노노그램 퍼즐로 변환합니다.
Puzzle List

퍼즐 목록

사용자가 생성한 퍼즐 목록을 확인할 수 있습니다.
Import Puzzle

퍼즐 뷸러오기

본인 혹은 다른 사용자가 생성한 퍼즐을 불러올 수 있습니다.
